Hướng dẫn biên dịch C++ trên Sublime Text 4

1. Tổng quan Sublime Text.

Hôm này mình sẽ hướng dẫn cách cài đặt môi trường để chạy C++ trên Sublime Text 3/4 cực kỳ đơn giản.

Đây là Text editor yêu thích cũng mình được vài năm rồi, mình trải nghiệm từ Sublime Text 2 và đến bây giờ là Sublime Text 4 với giao diện cập nhật cực kỳ đẹp.

Và khi nó được kết hợp các trình biên dịch với C++, python… thì nó trở lên quá tuyệt vời.

Nếu như chỉ cần làm các project đơn giản, giải quyết các bài tập, các thứ cần test với C++ thì mình luôn ưu tiên dùng Sublime Text vì thực sự nó giải quyết vấn đề quá nhanh chỉ cần mở lên code, ngay lập tức chạy được.

Với những ưu điểm:

  • Siêu nhẹ.
  • Nhanh.
  • Ổn định.
  • Free.
  • Đơn giản.
  • Tiện lợi.
  • Đẹp mắt.
  • Có các Package.
  • Snippset.

2. Cài đặt MinGW.

MinGW là một phần mềm mã nguồn mở, một trình biên dịch ngôn ngữ C/C++ trên Windows.

Các bạn tải trực tiếp tại link sau: minGW 17.1.

Tải về xong, ta mở lên chọn ổ C:\ và chọn Extract. Đợi một lát máy sẽ tiến hành cài, sau khi xong tiến hành bước tiếp theo.

Ta thêm minGW vào môi trường của Windows bằng cách:

Chọn ô tìm kiếm Windows hoặc nhấn tổ hợp Windows + S để hiện ra ô tìm kiếm, sau đó gõ tìm Environment và làm như video trên.

3. Cài đặt Sublime Text 4.

Các bạn tải trực tiếp tại link sau: Sublime Text 4. Chọn bản cho Windows tải về và cài đặt.

Sau khi cài đặt xong, các bạn mở Sublime Text lên. Ở trên góc trái chọn nút 3 gạch -> Tools -> Build System -> New Build System…

{
    "shell_cmd": "g++ -std=c++17 -Wall \"${file}\" -o \"${file_path}/${file_base_name}\" && start cmd /k \"$file_base_name\" && cmd /c exit && echo vietdo 2021 build system",
	"cmd" : "echo exit",
	"working_dir": "${file_path}",
	"selector": "source.cpp",
	"shell": true
}

Các bạn copy đoạn code ở trên và paste vào tab untitled.sublime-build vừa được mở ra.

Sau đó nhấn tổ hợp Ctrl + S và thay đoạn untitled trong untitled.sublime-build thành bất cứ tên gì mà các bạn thích ví dụ mình sẽ đặt là Run_Cplusplus.sublime-build sau đó save lại.

Bước cuối cùng. Ở trên góc trái chọn nút 3 gạch -> Tools -> Build System -> Tìm tên bạn vừa đặt lúc nãy và chọn là xong. của mình là Run_Cplusplus.

4. Chạy thử thôi nào.

Như vậy các bạn đã cài xong môi trường để có thể chạy C++ trên Sublime Text 4.

Mình sẽ tạo một file demo là helloworld.cpp sau đó viết một đoạn code c++ ngắn để test thử. Nhấn tổ hợp Ctrl + B để chạy để nhé các bạn.

Oke chương trình run thành công và không có lỗi gì và kết quả đúng như ý muốn của mình. Hehe quá dễ phải không nào. Chúc các bạn thành công nhé!!
