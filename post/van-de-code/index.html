Reading Binary File C++

  • Đỗ Quốc Việt

It extracts a given number of bytes from the given stream and place them into the memory.

void readingFile() {
	int n = 0;
	int *a;
	ifstream fin;
	fin.open("output.bin", ios::binary);
	if (fin.is_open()) {
		fin.seekg(4, ios::beg);
		fin.read((char*)&n, sizeof(n));	
	}
	fin.close();
}
